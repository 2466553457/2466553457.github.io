windows激活教程和工具

1.标明

注意：如果您的电脑系统是Windows10及以上版本，下载完成，解压本博客激活工具时，请先关闭Windows病毒防护系统！如果不知道怎么关闭请点击下方按钮

关闭病毒防护

2.教程 1

  1. 本教程仅适用于当下[2025-04-10]，当然不是说之后用不了，如果你是 2026 年看到的也可试一下好不好使。
  2. 本教程不仅可以激活 Windows 系统，还可以激活 office 软件。

第一步：查看自己电脑的系统是否为激活状态或是否有到期时间(如果显示未激活或某年某月某日过期，请继续看下方教程)，查看方法：按下键盘的 win + R ，在弹出的窗口输入：slmgr.vbs -xpr

第二步：右键桌面的 Windows 小图标[下图所示:]，如果你是 win10 系统就选择”Windows PowerShell(l)”,如果你是 win11 就选择”终端”(拿 win10 系统举例)。

第三步：在弹出的命令窗口中，写入：irm https://get.activated.win | iex ，点击回车。

第四步：弹出的窗口中，你只需要看第 1 项和第 3 项，这两项都是永久激活命令，第 1 项只能永久激活 Windows 系统，而第 1 项既能永久激活 Windows 系统，还可以激活 office 软件，(这里使用第 3 项命令做演示)按键盘上的数字3。

第五步：然后在弹出的窗口选择 5 即可！

第六步：然后稍作等待，一定要多等一会，(这里说一下，我的 win10 系统本就是激活状态，可能与你后续的显示代码不同，但无需担心，一切都是懒人步骤。)在弹出的窗口中按下键盘的数字0，等待一会，直到出现 Press any key to Go back…然后按任意键退出即可！

第七步：重复第一步骤，此时你就会发现系统和 office 都已经激活！

3.教程 2

如果 Windows 系统显示处于通知模式该如何解决？

这里我想说一句，前几天我遇到里一个问题，就是我刚组装的洋垃圾，装完老版本的 win10 专业版系统之后，发现用本文章下面的软件激活不成功，于是我在这里在告诉大家一个方法

1.首先，我们先查看一下 Win10 正式专业版系统的激活状态：按 win+r 键，弹出窗口，里面输入 slmgr.vbs -xpr，点击确定，查看系统的状态是什么时候到期或者是处于通知状态。

2.按 win+x 键，选择“命令提示符（管理员）”点击出现界面后，在此界面中，依次输出以下命令：

(1).输入 slmgr.vbs /upk 按回车卸载当前密钥（此时弹出窗口显未“已成功卸载了产品密钥”）。

(2).输入 slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX 按回车安装密钥（弹出窗口提示：“成功的安装了产品密钥”）。

(3).输入 slmgr /skms zh.us.to 按回车设置密钥管理服务计算机名（弹出窗口提示：“密钥管理服务计算机名成功的设置为 zh.us.to”）。

(4).输入 slmgr /ato 按回车激活产品，可能要等待 30 秒左右（此时将弹出窗口提示：“成功的激活了产品”，可能会没弹）。

3.按 win+r 键，弹出窗口，里面输入 slmgr.vbs -xpr，点击确定，查看系统的状态是否激活即可！

A.windows7 的常用版本分类

  1. Windows 7 Starter(简易版/初级版)

  2. Windows 7 Home Basic(家庭普通版)

  3. Windows 7 Home Premium(家庭高级版)

  4. Windows 7 Professional(专业版)

  5. Windows 7 Enterprise(企业版)

  6. Windows 7 Ultimate(旗舰版)

B.windows10 的常用版本分类

  1. Windows 10 Home(家庭版)

  2. Windows 10 Professional(专业版)

  3. Windows 10 Enterprise(企业版)

  4. Windows 10 Education(教育版)

C.windows11 的常用版本分类

  1. Windows 11 Home(家庭版)

  2. Windows 11 Professional(专业版)

  3. Windows 11 Enterprise(企业版)

  4. Windows 11 Education(教育版)

为什么要列举出 windows 版本？我想要告诉大家的是，在你做系统之前你电脑的系统是什么系统版本，就重新做什么系统版本，这样做完之后联网就会自动激活！

如果你想换一个版本或者不清楚自己的版本，请看下面用工具激活的教程！
