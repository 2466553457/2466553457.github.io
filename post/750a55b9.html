博客如何迁移

1.标明

注意：迁移到新电脑上之前您的新电脑必须是已经配置好环境了！

2.教程

这里只会教大家如何备份，安装环境很简单，大家可以去网上查找相应的教程！

第一步

首先备份博客，不需要全部备份，只需要图片中的框选部分!

第二步

然后在你的新电脑上新建文件夹命名为 hexo-blog(当然了名子随意嘛)，

第三步

然后将之前备份好的文件导入进去，

第四步

然后文件夹内空白处点击鼠标右键选择 Git Bush Here，

第五步

然后输入 npm install,来生成功能模块，

最后 hexo cl;hexo g;hexo s 或者 hexo cl;hexo g;hexo d。
